# Spark-Bank-System

## Introduction
>  
> A Spring boot web application used to transfer money between two users.

## Technologies Used:
>  ### Front-End:
- HTML
- CSS
- Bootstrap
> ### Back-End:
-  Java Spring boot 
-  MYSQL

### Flow of the website
Home > View Customers > Select One Customers > Transfer Money > See Transaction history
